# Materia Oscura

Repositorio para el análisis galáctico con datos del catálogo Gaia, centrado en estudios relacionados con materia oscura.

## Descripción

Este proyecto contiene scripts en Python para procesar y analizar datos astronómicos, principalmente relacionados con la dinámica y distribución de materia oscura en nuestra galaxia. El código principal se encuentra en la carpeta [`src/`](./src).
